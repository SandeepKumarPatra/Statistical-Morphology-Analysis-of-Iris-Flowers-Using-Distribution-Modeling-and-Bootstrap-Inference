Statistical Morphology Analysis of Iris Flowers Using Distribution Modeling and Bootstrap Inference

This project performs a deep statistical analysis of iris flower morphology using classical statistical methods.

The goal is to understand:

How flower measurements vary between species

Whether measurements follow Gaussian distribution

Whether species differ statistically (ANOVA)

Estimate confidence intervals using bootstrapping

Analyze skewness and kurtosis of features

Statistical Techniques Used

Descriptive Statistics

Gaussian Distribution Checking

Skewness

Kurtosis

Q-Q Plot

Bootstrap Confidence Interval

ANOVA (Hypothesis Testing)
